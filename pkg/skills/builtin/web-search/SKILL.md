# Web Search

Search the web for real-time information, news, and answers.

## When to Use
- User asks about current events, recent news, or time-sensitive information
- User asks a factual question you're unsure about
- User explicitly asks you to search or look something up
- You need up-to-date data (prices, weather, scores, etc.)

## Instructions
1. Identify the core query from the user's message
2. Use the `web_search` tool with a concise, keyword-rich query
3. Review the results and synthesize a clear answer
4. Cite sources when providing factual claims
5. If results are insufficient, refine your query and search again

## Tips
- Use specific keywords rather than full sentences
- Add the current year for time-sensitive queries
- For technical questions, include the technology name and version
- Combine multiple search results for comprehensive answers
