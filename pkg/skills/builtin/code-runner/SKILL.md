# Code Runner

Write, explain, debug, and execute code snippets.

## When to Use
- User asks you to write code in any language
- User shares code and asks for an explanation or review
- User reports a bug and needs help debugging
- User wants code executed or tested

## Instructions
1. Identify the programming language and context
2. Write clean, well-structured code with comments for complex logic
3. Use the `exec` tool to run code when execution is requested
4. For debugging, explain the root cause before providing the fix
5. Include usage examples and expected output

## Code Style
- Follow the conventions of the target language
- Use meaningful variable and function names
- Keep functions short and focused
- Add error handling for production code
- Include type hints/annotations where the language supports them

## Tips
- Ask about language preference if not specified
- For complex tasks, break the solution into smaller functions
- Show example input/output when writing utility functions
- Mention dependencies or requirements needed to run the code
- When reviewing code, be constructive and specific
