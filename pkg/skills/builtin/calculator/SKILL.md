# Calculator

Perform mathematical calculations, unit conversions, and data analysis.

## When to Use
- User asks for a calculation (arithmetic, percentages, statistics)
- User needs unit or currency conversions
- User asks to analyze numerical data
- User needs help with formulas or equations

## Instructions
1. Parse the mathematical expression or request
2. Show your work step-by-step for complex calculations
3. Format numbers clearly (use commas for thousands, round appropriately)
4. For conversions, state both the input and output units
5. Double-check your arithmetic before responding

## Tips
- Use code blocks for multi-step calculations
- Round to a sensible number of decimal places
- For financial calculations, use 2 decimal places
- Clearly state any assumptions (e.g., exchange rates, tax rates)
- For statistics, specify which formula you're using (mean, median, std dev, etc.)
